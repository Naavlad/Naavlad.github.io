Изменить отступ для дерева каталогов в пикселях

По умолчанию, отступ 8 пикселей.

Что бы изменить это значение, или добавляем в файл настройки:

// Отступ для дерева каталогов в пикселях
"workbench.tree.indent": 15,

Изменить цвета подсветки синтаксиса в VS Code

Customizing a Color Theme

Editor syntax highlighting

Чтобы настроить цвета подсветки синтаксиса редактора, используйте editor.tokenColorCustomizations в файле настроек пользователя settings.json:

Изменить для наиболее распространенных конструкций

Предварительно настроенный набор синтаксических токенов (“комментарии”, “строки”, …) доступен для наиболее распространенных конструкций

Изменить цвет комментариев в Default Dark+ :

"editor.tokenColorCustomizations": {
    "[Default Dark+]": {
        "strings": "#679267",
    },
}
Непосредственная настройка правил TextMate

Узнать значение scope для того элемента, который вы хотите изменить:
В командной строке(Ctrl+Shift+P) набрать Developer: Inspect Editor Tokens and Scopes

 "editor.tokenColorCustomizations": {
    "[Default Dark+]": {
      "textMateRules": [
        {
          "scope": "string.quoted.docstring.multi.python",
          "settings": {
            "foreground": "#616161"
          }
        },
      ]
    },
 }
