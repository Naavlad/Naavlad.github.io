PostgreSQL CLI
> CD C:\Program Files\PostgreSQL\14\bin
> psql -V # проверка версии
psql (PostgreSQL) 14.2
Изменение кодировки
> chcp 1251
Текущая кодовая страница: 1251
psql -U postgres
# изменение кодировки в консоли psql
postgres=# \! chcp 1251
postgres это пользователь, подключимся этим пользователем к дефолтной базе postgres

postgres=# \c postgres
Вы подключены к базе данных "postgres" как пользователь "postgres".

postgres=#

postgres=# create database shop;
CREATE DATABASE

postgres=# \c shop
Вы подключены к базе данных "shop" как пользователь "postgres".
shop=#
Основные команды
\q - Выход из консоли
\? - Справка по командам консольного клиента psql
\h - Справка по доступным командам SQL
\h <команда> - Справка по конкретной команде

\l - Список баз данных сервера
\c <dbname> - Выбор базы данных
\dt - Список таблиц в текущей базе данных
\du - Список пользователей
