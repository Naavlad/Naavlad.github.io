Установка Screen и .screenrc
sudo apt install screen
screen --version
>>> Screen version 4.09.00 (GNU) 30-Jan-22

# nano ~/.screenrc

# the following two lines give a two-line status, with the current window highlighted
hardstatus alwayslastline
hardstatus string '%{= kG}[%{G}%H%? %1`%?%{g}][%= %{= kw}%-w%{+b yk} %n*%t%?(%u)%? %{-}%+w %=%{g}][%{B}%m/%d %{W}%C%A%{g}]'

# no welcome message
startup_message off

# keep scrollback n lines
defscrollback 500

# 256 colors
attrcolor b ".I"
termcapinfo xterm 'Co#256:AB=\E[48;5;%dm:AF=\E[38;5;%dm'
defbce on

# mouse tracking allows to switch region focus by clicking
mousetrack on

# default windows
screen -t App  1  # default shell 
screen -t Database  2  # default shell 
screen -t bash  3 bash  # bash shell
select 0
bind c screen 1 # window numbering starts at 1 not 0
bind 0 select 10

# navigating regions with Ctrl-arrows
bindkey "^[[1;5D" focus left
bindkey "^[[1;5C" focus right
bindkey "^[[1;5A" focus up
bindkey "^[[1;5B" focus down
Команды Screen

Создать сессию с именем screen -S <имя сессии>
Подключиться screen -r <id>
Список сессий screen -ls

Cоздать новое окно Ctrl+a c
Переименовать окно Ctrl+a A
Просмотр списка созданных Ctrl+a w
Переключиться на окно №2 Ctrl+a 2
Удалить окно Ctrl+a k
Удалить все окна сессии Ctrl+a \

Разделить окно по вертикали Ctrl+a |

Выйти из screen Ctrl+a d
