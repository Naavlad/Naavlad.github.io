Как изменить shell в Linux

Одна из самых популярных командных оболочек — это оболочка Bash (Bourne Again Shell). Она используется по умолчанию в большинстве дистрибутивов Linux. Посмотреть командную оболочку у пользователя(у разных пользователей могут использоваться разные командные оболочки) можно разными способами. В переменной окружения. Переменная окружения SHELL хранит путь до исполняемого файла оболочки. echo $SHELL /usr/bin/zsh # для zsh /usr/bin/bash # для bash Через имя текущего работающего процесса Командой echo $0 входит в POSIX standards compliant shells .
